Entry Notes


Entry Notes (link) is a website I developed in ReactJS for the frontend and NodeJS for the backend. For styling, it uses MaterializeCSS, the login takes advantage of PassportJS, and the database is MongoDB (mongoose). This application will allow you to create a profile in order to store notes, documents, personal text, and other word-based data. In doing so, you can make public and private entries on your profile. The data will be digested such that it is easily accessible and easily searchable.

Mixture of Gaussians


Given a dataset of a population, this analysis can assume multimodal distributions. With this assumption, expectation maximization is done in R to fit a number of Gaussian distributions to the data. The visualization shows the counts along with the distribution overlayed, done in D3.

Population Distribution Visualization


This graph, created in D3, shows a basic visualization of grouped data. This displays 4 different data distributions among a population. The overlay of curves shows a (not necessarily accurate) assumption of an underlying normal distribution.

Automated Calculations Application


This application automated the calculations that the lab technicians in the weblab use to design their experiments. This allows users to input their desired parameters and the application itself performs all of the calculations associated with the chemical equations. There are also fields where the user can custom adjust their desired output, and the application itself still maintains the balance necessary to maintain accuracy. There are also custom exports for the application for Word and other data formats.

Gradient Descent Search Optimization


In my time at ENSCO, I designed and implemented a Gradient Descent algorithm to replace a brute-force approach at a search algorithm. The gradient descent algorithm performed exponentially better in time and had better results. Test runs that took 4 to 8 hours with the old algorithm, took 60 to 90 seconds with my implementation. The initial design was in R, but I re-wrote it in Fortran and implemented it in the backend of a tool.

Hierarchical Edge Building Analysis


This analysis uses either protein monomers, protein complex, or RNA data. Using R, a correlation between the respective data was found. Using the correlation data, the edges were drawn in D3 and displayed.
